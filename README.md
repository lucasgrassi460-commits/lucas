# Local WhatsApp Automation Platform

A self-hosted, locally running platform for WhatsApp automation with a web dashboard.

## Requirements
- Node.js (v18 or higher recommended)
- Internet connection (for initial npm install and WhatsApp connection)

## Installation
1. Install Node.js if you haven't already.
2. Open this folder in your terminal.
3. Run the following command to install dependencies:
   ```bash
   npm install
   ```

## Startup
1. Start the server using:
   ```bash
   npm start
   ```
2. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```

## Features
- **Dashboard**: View metrics, manage documents, bots, and device connection.
- **WhatsApp Integration**: Connect via QR code, persistent session, auto-reconnect.
- **Bot Logic**: Automatic replies based on Q&A match, fallback to bot instructions.
- **Documents**: Upload PDF/TXT context documents for bots.
